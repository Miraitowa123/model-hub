## Why

构建一个 AI 模型聚合平台的前端界面，让用户能够浏览、搜索和筛选全球优质 AI 模型。解决开发者难以发现和比较不同 AI 模型的问题，提供统一的模型展示和筛选体验。

## What Changes

- 创建完整的 React 前端工程，包含以下页面组件：
  - 顶部导航栏（Logo、导航菜单、搜索框、登录/注册按钮）
  - 主标题区（大标题、副标题、搜索输入框）
  - 功能按钮区（竞技场按钮）
  - 筛选栏（服务商、分类、能力、上下文长度四个维度）
  - 模型列表区（模型卡片展示，包含图标、名称、标签、简介、服务商、更新时间）
  - 列表顶部统计和排序选项
- 使用 Tailwind CSS 实现响应式布局和现代化 UI
- 模拟数据展示模型列表

## Capabilities

### New Capabilities
- `model-browser`: AI 模型浏览和搜索功能，支持关键词搜索和多维度筛选
- `model-card`: 模型卡片展示组件，统一展示模型关键信息
- `filter-system`: 多维度筛选系统，支持服务商、分类、能力、上下文长度筛选

### Modified Capabilities
- 无

## Impact

- 新增完整的 React 前端工程
- 依赖：React、Tailwind CSS、Lucide React（图标库）
- 无后端依赖，使用模拟数据
